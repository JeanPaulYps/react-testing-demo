# Ejecución del proyecto

Para ejecutar el proyecto correr:

```{bash}
$npm dev
$yarn dev 
```

# Ejecución de pruebas

Para ejecutar las pruebas correr:

```{bash}
$npm test
$yarn test 
```

# Obtener covertura

Para obtener la covertura de las pruebas correr:

```{bash}
$npm coverage
$yarn coverage 
```
